# 4by3
> :shipit: Returns the 4:3 or 16:9 ratio value of a requested number

## Install
```bash
$ [sudo] npm install -g 4by3
```
You might require `sudo` to install it globally.

## Usage
### 4:3
```bash
$ 4by3 800 # returns 600
```

### 16:9
```bash
$ 4by3 800 --wide # returns 450
```
